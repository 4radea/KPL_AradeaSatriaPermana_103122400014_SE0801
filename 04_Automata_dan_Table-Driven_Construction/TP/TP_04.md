## DESKRIPSI PROGRAM
Program ini merupakan aplikasi berbasis web sederhana yang digunakan untuk mengubah gaya teks menjadi huruf besar dan huruf kecil. Selain itu, program ini juga dapat menghitung jumlah huruf, huruf besar, dan huruf kecil secara otomatis saat pengguna mengetik. Aplikasi dilengkapi dengan fitur mode terang dan mode gelap untuk meningkatkan kenyamanan pengguna.
