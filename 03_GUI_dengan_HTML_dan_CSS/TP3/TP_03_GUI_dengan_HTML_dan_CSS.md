## DESKRIPSI PROGRAM
Program ini merupakan sebuah aplikasi web sederhana berbasis HTML, CSS, dan JavaScript yang berfungsi sebagai pengkonversi gaya teks. Pengguna dapat memasukkan teks ke dalam kotak input, kemudian sistem akan secara otomatis menghitung jumlah huruf yang diketik, termasuk jumlah huruf besar dan huruf kecil, tanpa menghitung spasi.

Selain itu, aplikasi ini menyediakan tombol untuk mengubah teks menjadi huruf besar, huruf kecil, atau format paragraf sesuai kebutuhan pengguna. Tampilan halaman dibuat sederhana dan rapi dengan menggunakan font Inconsolata dari Google Fonts serta tata letak yang berada di tengah halaman agar lebih mudah digunakan.

Note: Saya menambahkan fitur pada [index.js](index.js) agar teks yang dimasukkan pada program terdeteksi dengan baik dari fitur untuk mendeteksi huruf, huruf besar dan huruf kecil. Lalu, menambahkan fitur agar spasi tidak terdeteksi sebagai huruf
