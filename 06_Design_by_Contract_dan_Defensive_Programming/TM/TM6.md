## DESKRIPSI PROGRAM

Fungsi akan mengembalikan true jika angka tersebut bukan termasuk “fizz buzz”, dan false jika merupakan kelipatan 3 atau 5. Jika input bukan bilangan bulat yang valid, fungsi akan melempar TypeError.
