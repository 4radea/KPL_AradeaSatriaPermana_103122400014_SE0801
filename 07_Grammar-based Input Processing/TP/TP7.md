## Deskripsi Program
Fungsi toNumberArray digunakan untuk mengubah input berupa string atau array string yang berisi angka menjadi array angka.
Fungsi ini memisahkan nilai berdasarkan koma (jika string), menghapus spasi, mengonversi setiap elemen menjadi tipe number, dan mengabaikan nilai yang bukan angka.
