## Deskripsi Program
Program ini digunakan untuk menampilkan tanggal saat ini dengan format bahasa Indonesia, yaitu mencakup hari, tanggal, bulan, dan tahun (misalnya: “Selasa, 28 April 2026”). Program memanfaatkan Intl.DateTimeFormat agar format tanggal mengikuti standar lokal (id-ID),
sehingga dapat berjalan dengan benar di berbagai komputer tanpa perlu menuliskan format tanggal secara manual. Selain itu, hasil output diperbaiki dengan mengubah huruf pertama menjadi kapital agar tampil lebih rapi.
