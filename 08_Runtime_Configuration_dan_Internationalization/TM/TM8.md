## Deskripsi Program
Program ini adalah aplikasi sederhana berbasis Node.js yang digunakan untuk mengambil dan menampilkan kurs mata uang secara real-time. Program mengakses data dari API yang disimpan dalam file .env, kemudian mengonversi sejumlah nilai rupiah (IDR) ke dalam mata uang renminbi luar Tiongkok (CNH) dan euro (EUR).

Nilai hasil konversi ditampilkan dengan format mata uang dan tanggal yang rapi menggunakan Intl.NumberFormat dan Intl.DateTimeFormat. Program juga dilengkapi dengan pengujian untuk beberapa nominal, yaitu Rp25.000, Rp50.000, dan Rp100.000, sehingga pengguna dapat langsung melihat hasil konversinya secara otomatis.
